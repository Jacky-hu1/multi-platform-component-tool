# 元器件智能查询系统

基于GitHub Pages的元器件查询工具，完全免费，无需服务器。

## 🎯 四大核心功能

### 1. 💰 **价格对比**
- 多平台实时价格（立创、云汉、硬之城、华秋、贸泽）
- 价格趋势分析（最低价、平均价、最高价）
- 库存和交期信息
- 采购平台对比

### 2. 📊 **技术参数**
- **核心参数**：内核、主频、Flash、RAM
- **电气特性**：工作电压、温度范围、ADC分辨率
- **通信接口**：USART、SPI、I2C、USB、CAN
- **封装信息**：封装类型、引脚数量、尺寸

### 3. 🔄 **国产替代**
- **相似度分析**：95%、90%、85%等
- **价格优势**：-20%、-30%、-25%等
- **详细说明**：兼容性、差异点、适用场景
- **优缺点分析**：明确替代方案的利弊

### 4. 🔍 **智能选型**
- **综合评分**：基于价格、参数、替代方案的综合评分
- **推荐理由**：明确推荐使用的理由
- **注意事项**：使用中需要注意的问题
- **应用建议**：适合的应用场景

## 🚀 部署说明

### GitHub Pages部署
```
访问地址：https://jacky-hu1.github.io/multi-platform-component-tool/
```

### 部署步骤
1. **创建GitHub仓库**
   - 仓库名：`multi-platform-component-tool`
   - 描述：`元器件智能查询系统`
   - 选择：Public（公开）

2. **上传文件**
   ```bash
   git clone https://github.com/jacky-hu1/multi-platform-component-tool.git
   cd multi-platform-component-tool
   # 复制所有文件到仓库
   git add .
   git commit -m "部署元器件查询系统"
   git push origin main
   ```

3. **启用GitHub Pages**
   - 访问：`https://github.com/jacky-hu1/multi-platform-component-tool/settings/pages`
   - 分支：选择 `main` 分支
   - 文件夹：选择 `/ (root)` 根目录
   - 点击：`Save`

4. **访问网站**
   - 等待1-2分钟
   - 访问：`https://jacky-hu1.github.io/multi-platform-component-tool/`
